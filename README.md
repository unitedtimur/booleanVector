# booleanVector
1️⃣ 0️⃣  ⏩ The class is a boolean vector.  ⏪ 🇷🇺 🇺🇸

The class implements operations:
1. Disjunctions
2. Conjunctions
3. Inversion
4. Equivalence
5. Logical negation
6. Bit shifts
7. Weight of a vector
8. Size of a vector
9. Setting any bits to zero or one
10. Output and vector input
11. Copying
12. Getting any bit
